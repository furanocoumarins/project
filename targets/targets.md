# [Основная доска проекта](../README.md) 

|  				Мишень 				(белок) 			                              |  				Болезнь 				/ роль 			                       |  				Обоснование 			                                                                                                                                                                                 |   |   |
|-----------------------------------------------|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---|---|
| Ацетилхолинэстераза 				(AChE) 			                   | Альцгеймер 			                            | Классическая 				мишень для анти-альцгеймерских 				соединений (например, донепезил). 				Много работ по докингу природных 				соединений, включая фуранокумарины. 			                                           |   |   |
| Моноаминоксидаза 				B (MAO-B) 			                   | Паркинсон, 				депрессии, зависимости 			     | Фермент, 				регулирующий уровень дофамина и 				серотонина. Ингибирование MAO-B — 				стандартный подход в терапии Паркинсона 				(селегилин, разагилин). Есть данные о 				связывании фуранокумаринов с MAO-B. 			 |   |   |
| Дофамин 				D2-рецептор (DRD2) 				или NMDA-рецептор 			 | Зависимости, 				когнитивные расстройства 			 | DRD2 				— ключевой в механизмах зависимости 				и вознаграждения; NMDA — важен при 				нейродегенерации. Интересно проверить 				способность фуранокумаринов связываться 				с ними. 			                            |   |   |
